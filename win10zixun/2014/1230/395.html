Win10将加入大量全新功能回顶部

尽管官方口口声声说今年不再推出新版，可一个版本号为9901的Win10预览版还是抢在2014年的最后几天与我们见面了。在这个版本中，微软开启了传闻已久的桌面版语音助手Cortana（微软小娜），还解决了上个预览版中一个很重大的Bug——无法识别硬盘。虽然某些功能目前看起来还是存在着一些不足（比如小娜还不支持语音），但至少到此为止，围绕在Win10身边的几个最大悬疑都已经解开了。

那么问题来了，除了这些以外，Win10的肚里到底还有哪些没开启的干货？或许我们应该重新梳理一下，看一看这Win10到底还值得期待啥？

图01 Win10还能拿出什么干货么？

关注1. Win10版Aero

当年Aero的下课”，坊间一直有着各种传说。其实要小编自己来讲，原因莫过于两个，一是Aero的确耗电，不符合Win8续航至上的原则。二来嘛或许就是因为Metro刚上线，工程师们还不怎么习惯把这玩意儿和Aero融合到一起。

时光荏苒，两年时间过去，当初的Win8已经从服务平板，升级为平板+PC二合一。你平板在乎耗电，俺大PC可不在乎，犯得着也去享受这不美”之罪？此外这里还有一个更深层次的原因，那就是微软的老对手苹果，也在自家新版操作系统中加入了Aero，于情于理这Aero也该回归了！

图02 网上流传的疑似Xbox One泄漏图（PS：Xbox One未来也将由Win10驱动）

当然如此重要的盛典”显然不能将Win7版Aero直接搬回来了事，毕竟多了一个Metro视图不是。那么Win10版的Aero到底会是什么样？没人知道，不过前不久一则Xbox One Dashboard主屏泄漏图或许可以给你我一点点思路，类Metro屏幕，蓝色半透明磁贴。最关键的是，Xbox One未来也会由Win10进行驱动。当然这里顺便也和微软提个醒吧，Aero还有磨砂更带感些，单纯的半透还是太单薄了！

关注2. 新资源管理器

资源管理器已经沿用很多年了，即便是历经XP、Vista、Win8这几次大版本更新，也没有出现太大改动，不过这并不意味着Win10也会对它无动于衷。

日前微软Windows Insider计划负责人Gabriel Aul的一则推特消息，让我们相信Win10仍在改进它的资源管理器。

酷！资源管理器体验相关的改进还在进行，用不了多久你就能看到一些极好的东西出现。”

图03 网友设计的新资源管理器畅想图

问题是，什么才算极好”？新版资源管理器又会改进哪些地方？在一张网友设计的畅想图中，我们或许能得到一点点感悟。在这张图中，地址栏和文件层级被明显淡化，搜索栏也被挪到了窗口左侧，整个画面看起更加协调。按钮化和扁平化是这张图片的主思路，看得出是专门为触屏用户优化的。最为关键的是，桌面版用户仍可按照以前习惯操作这些功能，如果说唯一与现版不同的，那就是没有专门的快捷按钮区了。

关注3. 多标签文件管理

除了全新UI，多标签也是很多人关心的一个话题。之前微软已经在IE浏览器中实现了这一功能，这次很可能会加入到资源管理器中。

图04 Q-Dir的多窗口思路是否是微软的下一个方向呢？

目前第三方管理器中实现多标签浏览的已经很多，因此我们并不难想像未来的微软版”会是一个什么样子。但就小编个人来讲，多标签的便利性显然不如多窗格，毕竟很多情况下资源管理器还是以提供文件操作为主。

从这个角度上考虑，似乎Q-Dir的设计更贴切一些。当然目前的Q-Dir风格还是和Win10不太搭调，主要是界面过于繁杂且对初学者难度较高。但如果微软就是确定了这个思路，那么改进就是在情理中了，毕竟方便与美观，这个理儿大家还是拎得清的！

关注4. 新系统时间

右下角时间区已经沿用了近二十多，甭说我们，就连微软恐怕都有些看不下去了。那么这Win10会不会对系统时间做一个调整呢？

图05 Build 9901中隐藏的新时钟模块

最新版Win10 Build 9901为我们打开了一个全新思路。在这款全新的预览版中，我们可以通过注册表开启一个隐蔽模式下的系统时钟。在这个界面下，你会看到一个全新的时钟面板，涵盖日历、时钟、闹钟三个部分，而且风格也和新Windows倡导的扁平、轻快颇为统一，视觉感相当不错。

关注5. 可交互磁贴

由图标发展至磁贴，可以说是Windows历史上的一次进步。道理很简单，它能将信息直接显示在磁贴上，从而改变了之前打开软件才能看信息这个流程”。当然我们知道，微软的心肯定不止这么一点儿。在Win10还被称为Win9的那个时代，网络上就曾经爆出过一则有关Windows下一代可交互磁贴的消息。其中最引人注目的一个变化，就是磁贴中被加入了一些可交互式功能，比方说直接删除一封电子邮件，或者拖拽查阅新闻列表等等。

图06 早期流传的可交互式磁贴示意

随着新版Win10消息的不断增多，有关下一代爆裂式磁贴（Mixview）的新闻也开始浮出水面。和可交互式磁贴类似，爆裂式磁贴的精髓也是为磁贴加入一些可操作模块。只不过前者是通过一些标准化操作（如点击、拖拽、右键等），而后者更倾向于用右键点击来实现各类功能。目前看两种模式在Win10中似乎都有可能实现，也许就是一个平板版一个桌面版之分吧。


图07 这是Windows Phone 10中已经曝光的爆裂式磁贴”

关注6. 配置云备份

一直以来，系统重装都是件让人头疼的事，不光是PC，手机端也是类似的问题。一般来说以目前的电脑速度，安装一套系统最多也就20多分钟时间，可安装各类软件以及将软件设置成自己喜欢的模式，则很可能消耗你半天以上的工夫。

当然你肯定会想到目前一些手机管理软件所提供的数据备份功能，甭管多少软件，只要提前用它备份一下，装好系统后把它们恢复回来，即可得到和原来一样的体验，甚至连之前的搜索关键字和游戏进度也不会丢失。

图08 Build 9901中的OneDrive Backup”面板

越来越多的迹象表明，Win10似乎也开始要享受这样的待遇了。在最新泄漏的Build 9901预览版中，你会发现设置”→更新和恢复”标签下多出了一个OneDrive备份功能。内容涵盖开始菜单列表、密码、收藏及其他系统设置。当然目前这项功能也和微软账户同步一样，无关第三方软件，但在这个时时求变的时代，谁能保证微软不会动这个念头呢？

关注7. 自适应机制

One Platform是微软对于Win10的定义，什么PC呀、平板呀、手机呀、ATM等等，这么说吧，但凡有屏幕的，估计微软都想插上一脚。当然这里会涉及到一个技术问题，即不同的设备所需要的操纵是不同的。比方说PC用户可能更青睐于键盘+鼠标，平板用户则更关注于触控方面改进的怎样，至于手机嘛则是尽可能在最小的空间内显示出尽可能多的信息。总之不同的设备总会有自己所不同的诉求点，这其实就是最难权衡的。

图09 这个曾被认为是Continuum版开始菜单，意思是Win10的开始菜单仍会改？

Continuum是Win10中一项很重要的特性，原理很简单，就是通过检测电脑上（这里是指广义概念的电脑）不同的输入设备，来智能决定使用何种屏幕布局及界面。一个最简单的例子是，当你插入键盘时，Win10会认为你是键鼠用户，从而默认显示经典开始菜单（新版）及标准屏幕布局。而拔下键鼠使用触摸屏时，则会显示开始屏幕及放大布局，毕竟要给手指留下足够的地方。如果你觉得上面的介绍过于笼统，那么一张来自微软的专利图或者更能说明问题，对比很明显。

图10 微软曝光的一款自适应机制专利图

关注8. 高级通知中心

Build 9860版首次开放了通知中心，当然目前这项功能还显得有些初级，仅仅就是一个通知”罢了。那么微软眼中的通知中心到底会是什么样呢？或许外媒NPU的一条消息，可以给我们一点点思路。

……不仅有可操作通知，而且现有的通知+操作中心的UI也会进行革新，将会出现更多的控制选项，比如音乐控制等……”

图11 目前手机下拉栏或许是未来Win10通知中心的一个榜样”

没错！听起来就像是手机里的下拉状态栏。总之未来的通知中心应该就像磁贴一样，更加突出交互及可操作性，而不仅仅是通知”了事！

关注9. 堪比小娜”的Skype同声翻译

小娜”的出现终于可以让语音第一次光明正大地控制电脑，然而微软的野心”似乎并不止于此。日前一则介绍微软神经网络语音识别技术的视频现诸网络，从视频上看，微软似乎已经在着手准备另一项基于语音的技术——Skype同声翻译。

图12 这是微软有关Skype同声翻译的视频截图

这项功能与电视里经常见到的同声翻译有着很多相似之处，同样可以保证用户在通过Skype一对一交流时，实时听到熟悉的母语。所不同的是，专业的同声翻译是由人工实时完成，而Win10则是由机器自动翻译。

从目前汇总到的资料来看，Skype同声翻译可以支持语音、视频两种模式。当然基于目前的机器翻译能力，翻译过来的语言或许会有不通甚至听不懂的情况。但我想既然我们能对整句翻译存在较大宽容，那么对于这项技术就不应抱着太多质疑。

写在最后

其实类似的话题在网络上还有很多，从某种意义上讲，微软在更换了CEO后的确增加了很多新气象。总之甭管未来的Win10走向何方？至少我们看到它一直还在努力着。或许以后的世界真的不是某家公司一家独大的时代，但这又有什么不好呢？
