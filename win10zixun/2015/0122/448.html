微软发布Win10消费者预览版 贯穿全平台

http://www.win10os.com win10系统 发布时间：2015-01-22 13:40 来源：win10系统 浏览：加载中次

北京时间1月22日凌晨消息，微软在西雅图雷蒙德总部正式发布了新一代操作系统Windows 10的消费者预览版。

微软发布Win10消费者预览版

微软Windows主管Terry Myerson率先登台，称Windows 10将比以往更好的保护合作伙伴数据，通过Windows 10，Windows“通用应用”将被带到Xbox One上，“Windows 10将致力于各个设备间的交互，从PC到手机到平板到游戏机，无论是内容还是操作方式。”

Windows 10横跨多平台

Windows 10发布的第一年内，所有Windows 7、Windows 8.1和Windows Phone 8.1用户将免费升级最新版系统。用户一旦升级至Windows 10，将会获得永久的升级更新服务。Windows 10消费者预览版的PC版本将在下周面向订阅用户推送升级，而手机版将在2月到来。

Myerson表示，用户、开发者为微软开发Windows 10提供了很多建议，Windows 10将是史上支持最兼容的操作系统。Windows 10的开发人员表示，全球有15亿设备搭载Windows系统，Windows 10的开发和完善建立在用户反馈的基础上，他们在全球约260个地区进行了用户测试。员工们还表示，感谢用户的反馈，将会以最好的结果回报他们。

Windows 10的全新界面

　　随后，微软Windows Phone企划管理部企业副总裁Joe Belfiore演示了最新版Windows 10的一些功能特性。新系统的活动中心不仅可以推送通知，还拥有WiFi、蓝牙、屏幕亮度、设置、飞行模式、定位等快速操作选项。

　　Windows 10采用了全新的开始菜单、开始屏幕和CharmBar，以及动画效果。当退出纯触控模式后，任务栏会出现，应用会以窗口的形式呈现。

微软Windows Phone企划管理部企业副总裁Joe Belfiore

对于混合（二合一）设备来说，Windows 10可以提醒用户选择使用模式，在平板和桌面模式之间切换。桌面版用户将找到操作鼠标时的使用体验，以及更多的动态磁贴和右侧的活动中心。同时，8英寸设备也将拥有不同的全新体验。

正如之前的传闻，微软语音助手Cortana正式登陆Windows 10，虽然是早期版本，但是识别的成功率和理解能力依然很强，并且可以了解用户的喜好和习惯。Cortana位于Windows 10的任务栏上方，随时可以启动搜索，搜索的内容包括本机硬盘、OneDrive云端文件等各类信息。演示过程中，Belfiore成功利用Cortana实现了播放音乐、撰写电子邮件等操作。

活动中心的快捷设置

Windows 10在手机和平板上同样具有很好的体验，手机版系统的操作中心界面定制性更强。同时，“超长”的设置列表也进行了归类。新系统支持首页壁纸更换，不再是纯黑或纯白的背景。此外，通知中心与PC实现同步，用户可以在其他设备上继续处理未在PC上完成的工作。当有通知提示时，可以下拉进行更多操作，例如快捷回复短信等。有意思的是，输入键盘可以全屏移动，解决了大屏手机的操作难题。

Belfiore称，Skype已经深度整合至Windows 10，用户可以在短信应用内直接使用Skype聊天，也可以随时切换到短信。

Windows 10移动版

输入键盘可以全屏移动

语音助手Cortana

在“通用应用”中，Windows 10触屏版将内置专为触控设备设计的全新Office，包括Word、Excel和PPT。横屏之后，操作界面与桌面版应用非常类似，功能也更加丰富，应用切换更流畅，并且支持无线打印。

触屏版Office

同时，Windows 10将具有全新的Outlook客户端，其内置完整的Word编辑功能，可以修改文本内容和电子邮件。在邮件列表中，向左滑可以快速删除，向右滑可以快速旗标。

Cortana查询天气

此外，“通用应用”还将包括日历、照片、人脉、音乐、地图等。其中，图片应用支持自动美化照片，可以自动生成相册，相册基于云端分享。全新的浏览器内部代号“Spartan”，其不仅采用了全新设计界面，引擎也是新的，带来了卡片式体验，并且整合了Cortana。

使用过程中， 用户可以直接将网页作为“画布”进行手写标注，支持手指和触控笔输入。标记完之后，可以收录进OneNote，选择与其他人分享。而在阅读模式中，用户则可以更专注于文字浏览，微软为此也提供了一个云端同步的阅读列表。

“Spartan”浏览器画布标注

既然是全平台，Xbox也是微软生态的重要组成部分。Xbox和游戏部门主管Phil Spencer介绍称，Windows 10为Xbox带来了统一体验。新系统集合了Xbox应用的各种功能，包括好友列表、游戏成就、视频录像等，就像是Smart Glass的完整版。Windows 10上的游戏将可以完成最后30秒的视频录制，并切换到游戏应用中进行视频分享。

Xbox和游戏部门主管Phil Spencer

Xbox游戏界面

在“我的游戏”中，玩家可以看到自己所有设备上的游戏，以及所有好友的情况。新系统还内置跨平台的语音、文字聊天功能。同时，Direct X12还将支持Unity游戏引擎。据了解，DirectX 12较当前的性能提升了50%。值得一提的是，Xbox能够以视频串流的形式将游戏运行在PC和平板上。未来，Windows 10将是Xbox One的操作系统。

游戏效果提升

Fable Legends游戏演示

全新硬件：Microsoft Surface Hub一体化电脑。这款产品配备84英寸4K显示屏，支持多点触控和手写笔输入，采用了先进的传感器，双摄像头，运行Windows 10操作系统。Microsoft Surface Hub可以应用于集体会议，可以实现多人操作和共享同一个文件或应用，共同编辑。微软并未提及这款产品的售价，以及娱乐功能方面的应用。值得一提的是，Microsoft Surface Hub的主要研发人员是中国人。

Microsoft Surface Hub配置信息

Microsoft Surface Hub使用演示

来自Kinect部门的Alex Kipman展示了微软最新的增强现实技术Windows Holographic，并且发布了虚拟现实头罩Microsoft HoloLens。这款设备采用半透明眼镜片设计，内置处理器和高级传感器，支持三维声音，无需与手机、电脑连接使用。

Microsoft HoloLens

现场演示过程中，使用者佩戴Microsoft HoloLens可以看到墙上的操作界面，对着“空气”做出一系列操作，大屏幕显示出使用者本人能够看到的“悬浮在空中的物体”。使用者可以对这些物体进行手势或声音控制操作。经过几步操作，使用者直接设计出了一架小型无人机，并且通过3D打印制造了出来，非常具有科幻色彩。

使用者隔空操作

3D打印出的小型无人机

发布会的最后，微软CEO萨提亚·纳德拉进行了总结：“今天无疑是个大日子，无论是对消费者、合作伙伴还是商用用户。微软希望创建一个移动至上，云端优先的世界，这一切的核心，就是Windows10。”

微软CEO萨提亚·纳德拉

纳德拉称，Windows 10并不只是微软的新产品，它更代表了微软的野心和愿景，微软希望用户能够爱上Windows，该公司正在整合一切资源，帮助用户的做得更多，“这就是跨平台的移动体验，这是微软专注的东西。”

微软CEO萨提亚·纳德拉

虽然Windows 10看起来更像是“回归传统”，但伴随着微软“移动优先 云优先”的策略，这一系统在移动平台的表现将更为关键。纳德拉领导下的微软，将在Windows 10上实现真正的全平台：开发者编写一次代码，带来不同平台的统一体验。
