更懂中文了 Win10 Build 14986版发布

“休息”了三周后，Win10重新迈开更新的脚步，发布了Windows 10 Build 14986版，根据微软官方的说法，Win10 14986版对中文的“理解”更好了，不仅小娜(Cortana)可以帮助识别歌曲，中文输入法也更友好了。

Windows 10 Build 14986新功能：

Cortana（小娜）更棒了

现在小娜可以在中国为用户识别音乐了：只要点击小娜主页右上角的音符图标，小娜就会通过麦克风接收正在播放的音乐，然后通过网易云音乐来比对并告诉用户正在播放的歌曲信息。如果用户开启了“你好小娜”的响应命令，还可以直接发出语音命令“你好小娜，这是什么歌”让它帮你查找歌曲。

你好小娜，这是什么歌

语音控制关机、重启、锁定和睡眠：Cortana现在可以正确执行诸如“关机、重启”之类的语音命令了，目前支持的命令有：关机、重启、锁定、计算机睡眠、增加音量、减小音量。

启动第三方音乐应用播放歌曲：英文系统的Cortana现在可以控制更多音乐APP来播放歌曲了，最新加入支持的APP有 iHeartRadio 和 TuneIn Radio，用户可以向Cortana发出类似“Play <艺术家/曲目/标题/类型> on <应用名称>”这样的语音指令来实现音乐播放。同样，用户也可以直接用语音命令来调节音量。

设备空闲时小娜将以华丽的全屏形态亮相：当用户设备处于空闲状态时，使用“你好小娜”激活并向它发出某些指令会看到一个全屏化的Cortana界面。

全屏下的Cortana界面

允许用户以Azure Active Directory身份登录Cortana。

中文输入法改进

拼音候选词支持智能模糊匹配：作为提高中文输入效率的一个重要改进，现在不仅能显示模糊音候选词，更会明确标注候选词的确切拼音。

拼音输入法的智能模糊匹配

除了模糊音智能匹配外，状态栏的输入法图标有了全新的右键上下文菜单；自学习短语现在支持导出和导入；微软五笔输入也开始支持用户自定义短语的添加、导入和导出了。

中文（简体）手写输入改进

14986版现在能够识别用户连续写出的一整行文字了，无需再像之前版本中一个字一个字的识别。而当用户发现写错某个字时，只需在错字上从右向左划一道线即可删除这个字，且其他已写入的字不受影响。

中文手写更好用了

手写输入时还有一个容易遇到尴尬状况是，某些左右结构汉字在手写时由于间隔稍大被系统判定为两个字，一般情况你只能删除后重写，但现在你可以通过一个特定操作做告诉系统这其实是一个字，那就是在分开的两半字外划个圈。这个功能同样适用于将两个单独的字组合成一个字，比如人人成从、木木成林等。

文字组合

Game Bar支持更多游戏，该版本Win10增加了对近20款全屏游戏的支持，在这些游戏中用户都可以按下【Win + G】来激活Game Bar来进行截图和游戏画面录制。

Windows Ink体验进一步改善，打开屏幕草图后可通过工具条“恢复上一个”功能继续编辑上次的草图；一些选项和工具的视觉效果改进。

Edge浏览器新增数个扩展，包括记录常用密码的True Key，网购相关的Ebates Cash Back，以及网页笔记类的Read Write。

UWP应用内容渲染方式改进

讲述人功能改进

新增Windows Defender仪表板应用，方便用户了解和掌控设备的安全和运行状况。目前该应用仅支持英文界面，并且只是早期形态。

新增的Windows Defender仪表板

注册表编辑器改进，添加了和文件资源管理器相类似的快捷键导航，【Alt + 上】回到上一级项，【Alt + 左】后退，【Alt + 右】前进。另外还加入了字体调节功能，用户可以更改注册表编辑器的文字显示效果了。

USB Audio 2.0设备将暂时强制使用微软提供的驱动而非设备专用的第三方驱动

改善Win10更新体验，用户可以主动决定系统何时安装更新，避免因执行更新带来的无法正常使用计算机的影响。

其他一系列的Bug修正和性能改进

Windows 10 Build 14986已知问题：

·键盘快捷键调节屏幕亮度仍然有问题，暂时需要通过操作中心或者设置-系统-显示里的选项来调节；

·打开OneDrice文件夹可能会收到一个错误提示对话框，若遇到该问题可通过以下方法来解决。打开管理员级别的命令提示符或Power Shell窗口，输入：attrib -O <弹出错误提示框的文件夹完整路径>

·某些Windows Insider用户可能无法使用“直接访问（DirectAccess）”功能；

·刚安装完音乐APP，“Hey Cortana, playon”的语音命令可能无法正常工作，需要稍等上几分钟再次尝试。

·Windows Defender仪表板目前只有最基础的反病毒和防火墙信息，且只能查看。
